Sleep Gang

This project's main objective is to monitor a person's sleeping habits. Temperature, humidity, luminosity, and a person's motion during sleep and the length of time they slept are the factors that we used to analyse a person's sleep pattern.

To track a person's sleep, we are using an esp32 with sensors and periodically transferring data to OM2M. The data is then extracted from oneM2M using Python and Flask, and the data is then analysed. Additionally, you are accessing our analysis right now using a cloud-based solution that we have developed.
